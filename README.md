# Automotive Face Detection Attendees System  ## Overview This project is a real-time face detection system designed for attendee verification in automotive or event settings. Implemented in Python using OpenCV, it uses a machine learning-based Haar Cascade classifier to detect human faces from a webcam video feed.
## Features
- Detects faces in real-time from webcam input.
- Highlights detected faces with bounding boxes.
- Suitable for attendee monitoring and event management.
- Easy to run and extend.

## Requirements
- Python 3.7 or higher
- OpenCV (install using pip install -r requirements.txt)
- Webcam or video capture device
